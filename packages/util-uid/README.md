# @onflow/util-uid

Utility for creating Unique Identifiers.

# Status

- **Last Updated:** July 17 2020
- **Stable:** Yes
- **Risk of Breaking Change:** No

# Install

```bash
npm install --save @onflow/util-uid
```

# Usage

```javascript
import {uid} from "@onflow/util-uid"

const uniqueIdentifier = uid()
```
